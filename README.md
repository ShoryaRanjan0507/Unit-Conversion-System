# Unit-Conversion-System
A beginner friendly Python programme for converting units like length, weight and temperature.
A simple Python console-based Unit Converter that helps users convert Length, Weight, and Temperature values quickly and accurately.

# Features

* Length Conversion
- Kilometers ↔ Meters
* Weight Conversion
- Kilograms ↔ Grams
* Temperature Conversion
- Celsius ↔ Fahrenheit

# Technologies Used

* Complete Details:

- Language -> Python
-  Interface -> Console/Terminal
- Concepts Used -> Input/Output, Functions, Loops, Conditional Statements
